首先，是很忙的一周暂时告一段落，截止周二，这一连上的周有三场期中考试、一场物化 pre，还抽出时间做 LiuGroup 这边的事情
其次，五一假期很完美。
第一天早上 8-10 羽毛球（我、郭坤硕、闫博文、群约大佬），郭和大佬九点才来，大佬可能嫌强度不高，打了一会就走了。
之后，我和郭稍加考虑，决定说走就走骑车去碧海金沙。大约上午十一点十分出发，晚上九点才返回。如果只有我自己可能会更快，不过我第二天屁股也有点疼。看来之前还是对自己有点乐观了，单日单飞 200km 对现阶段的我来说还并非一件轻而易举的事情。
第二天去江湾待了一天，把思绪基本全部整理好了。学习了 ASE、GraphiQL 和 CatalysisHub 的 api 使用
第三天上午去江湾游泳，中午爸爸妈妈就来了。我们转了转江湾校区，又回到酒店玩了玩 switch。他们感觉不错，都觉得很有意思的样子
第四天本来说去野生动物园，但我又感觉好麻烦，不如躺一天（）。上午九点多去，玩了玩，和妈妈一起看了好多 switch 上可以玩的游戏，到中午吃了买的蛋糕。然后一起去看了电影《哈尔的移动城堡》，很美丽的一段童话。看完电影出来大概三点，去百联（电影就在百联）楼下的达乐美吃了一顿，转了转，他们就走了。
第四天也是我的生日。三妈和大双祝福、发红包，我都没要。晚上发了个朋友圈。
第五天基本就在寝室摆，晚上写了分析化学作业——之后就开学了。
