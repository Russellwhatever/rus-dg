# 对次表面金属其实是有讨论的
文章认为，金属倾向于存在于次表面的情况有三类：
1. 非互溶金属，其中 **solute 金属的表面能远大于 host 金属**。如 Fe/Cu (111) 和 Co/Cu (111)。
这部分内容在 Sec. VIII 讨论
2. 互溶金属，彼此之间可以和进化，并可以形成有序结构。如 Pd/Cu (110) 和 Au/Ag (110)（参考文献：50,51）
